## 🚀 快速开始

### 前置要求

- Node.js 18+
- npm 或 yarn

### 安装

1. 克隆仓库
```bash
git clone https://github.com/EXLF/AI-Airdrop.git
cd AI-Airdrop
```

2. 安装依赖
```bash
npm install
# 或
yarn install
```

3. 启动开发服务器
```bash
npm run dev
# 或
yarn dev
```

4. 打开浏览器访问 `http://localhost:3000`

## 🔧 构建

```bash
npm run build
# 或
yarn build
```
